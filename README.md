# income-genie

## Что мы знаем о признаках?
- Одна строка - это множество характеристик, закрепленных за **одним** клиентом (его id, gender, age, w);
- Прогнозируемый показатель (target) - доход клиента (а за какой период вообще?);
- Все прочие показатели, которые связаны с деньгами и кредитами, описывают денежную активность клиента за **разные** промежутки времени (какие-то признаки посчитаны за полгода, какие-то - за год);
- Дата актуальности признаков - это дата, когда набор характеристик был зафиксирован в таблице (когда, условно говоря, сделали выгрузку таблицы из какой-то системы);
- Есть группа данных, которая относится к **цифровому профилю** клиента (признак начинается с *dp_*);
- Есть группа данных, которая относится к сведениям клиента из **бюро кредитных историй** (начинается с *hdb_* и *bki_*);
- Есть группа данных, которая относится (предположительно) к статистике клиента из внешних систем (начинается на *vert_*);
- Есть группа **средних** данных, посчитанных за разные периоды (начинается на *avg_*, но не всегда);
- Остальные данные (по крайней мере, их наименования) разрозненны.

## Поднять композ

```bash
docker compose up --remove-orphans -d
```
